## About SeverQuest
This is a local online quest system made with Laravel + Vue.js.

## Used Technologies
- Laravel 5.7
- Laravel Echo
- Telegram Bot SDK
- Vue.JS (with Vuex)
- Vue Notification
- Vue Chart.JS
- Pusher
- Dashio Dashboard
